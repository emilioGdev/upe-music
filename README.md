# 🎵 UPE Music

Projeto desenvolvido como parte da disciplina **Gerência de Configuração** da Universidade de Pernambuco (UPE). O objetivo é criar uma aplicação web simples com estrutura de front-end e back-end separados, simulando uma plataforma musical.

---

## 📁 Estrutura do Projeto

```
upe-music/
├── front-end/      
├── back-end/        
└── README.md        
```

---

## 🚀 Tecnologias Utilizadas

### Front-end
- HTML5
- CSS3
- JavaScript

### Back-end
- Node.js 
- Express.js
- SQLite

---

## ⚙️ Como Executar o Projeto

> Requisitos: Node.js e npm instalados

### 1. Clone o repositório

```bash
git clone https://github.com/emilioGdev/upe-music.git
cd upe-music
```

### 2. Instale as dependências 
```bash
cd back-end
npm install
```

### 3. Inicie o back-end

```bash
npm start
```

### 4. Abra o front-end

Você pode abrir o arquivo `index.html` diretamente no navegador ou servir com um servidor local:

```bash
cd front-end
npx live-server
```
